# SimpleRelics

SimpleRelics is a Vanilla+ mod that adds simple, single-use relics that prevent death and grant small, balanced effects.

The goal is to enhance survival gameplay without adding complex RPG systems or overpowered mechanics.

But be careful, they will consume you...
